= Custom OpenShift Virtualization lab

== Preparation steps

1. Upgrade all operators
2. Define ocs-external-storagecluster-ceph-rbd-immediate StorageClass as default
3. Add htpasswd file
4. Create specific role
6. Create namespaces project-1 and udn-projet-1 with associated role
7. Ajouter le role kubevirt.io:migrate aux utilisateurs

== Create namespace

----
for i in {1..20}
do 
oc new-project project-$i
oc create ns udn-projet-$i --dry-run=client -o yaml | oc label -f - --local k8s.ovn.org/primary-user-defined-network="" -o yaml | oc apply -f -
oc adm policy add-role-to-user admin user$i -n project-$i
oc adm policy add-role-to-user admin user$i -n udn-projet-$i
oc adm policy add-role-to-user admin user$i -n openshift
oc adm policy add-role-to-user kubevirt.io:dm user$i -n project-$i
oc adm policy add-role-to-user kubevirt.io:dm user$i -n udn-projet-$i
oc adm policy add-role-to-user edit user$i -n openshift-migration
done
----
