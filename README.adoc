= Custom OpenShift Virtualization lab

== Preparation steps

1. Upgrade all operators
2. Define ocs-external-storagecluster-ceph-rbd-immediate StorageClass as default
3. Add htpasswd file
4. Create specific role
6. Create namespaces project-1 and udn-projet-1 with associated role
7. Ajouter le role kubevirt.io:migrate aux utilisateurs

== Create namespace

----
oc adm policy add-role-to-user admin user01 -n project-1
oc adm policy add-role-to-user admin user01 -n udn-projet-1
oc adm policy add-role-to-user admin user01 -n openshift
oc adm policy add-role-to-user kubevirt.io:dm user01 -n project-1
oc adm policy add-role-to-user kubevirt.io:dm user01 -n udn-projet-01
oc adm policy add-role-to-user edit user$i -n openshift-migration
----
